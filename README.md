# Neurostore + Neurosynth Compose

<p align="center">
  <img src="compose/neurosynth-frontend/public/static/synth.png" width="180" alt="Neurosynth Compose logo" />
</p>

This monorepo contains the backend and frontend services that power the Neurosynth ecosystem:

- `store/`: **Neurostore** backend (study, image, and metadata storage APIs)
- `compose/backend/`: **Neurosynth Compose** backend (project workflows and execution)
- `compose/neurosynth-frontend/`: **Neurosynth Compose** web application

Neurosynth Compose is a free, web-based platform for reproducible neuroimaging meta-analysis, including study search, curation workflows, extraction, and cloud/local analysis execution.

## Paper

If you use Neurosynth Compose in your research, please cite the following paper:
> James D. Kent, Nicholas Lee, Angela R. Laird, Taylor Salo, Julio Peraza, Katherine L. Bottenhorn, Kendra Oudyk, Thomas E. Nichols, Jean-Baptiste Poline, Alejandro de la Vega; Neurosynth Compose: A web-based platform for flexible and reproducible neuroimaging meta-analysis. Imaging Neuroscience 2026; 4 IMAG.a.1114. doi: https://doi.org/10.1162/IMAG.a.1114
